Screen 1 · Home Feed

  • Day strip = week-at-a-glance with zero taps
  • Left accent bar on card = instant status read
  • Join CTA in every card — no detail page required to act
  • Player avatars communicate social proof immediately

Screen 2 · Create Match

  • 3-step flow prevents form fatigue
  • Progress bar = orientation without anxiety
  • Time slots replace freeform input = fewer errors
  • Format cards = scannable, never dropdowns
  • Summary card on step 3 = confidence before submitting

Screen 3 · Match Details

  • 3-tab structure: Players / Chat / Info = zero overlap
  • Confirmed vs Waiting groups are clearly separated
  • Chat co-located with match context, not a standalone app
  • Sticky CTA always reachable regardless of scroll depth

Screen 4 · Profile

  • Reliability ring = trust signal, visible before any action
  • Stats grid mirrors Strava's clarity (number + label)
  • Activity chart shows consistency, not just totals
  • Recent matches = proof of attendance pattern

Screen 5 · Discover

  • Map first = spatial context for local decisions
  • Segment tabs: Matches / Players = single focus at a time
  • Filter chips persistent + scrollable = no modal friction
  • Distance-first sorting = highest relevance for urban users

UX principles applied

  • 8pt spacing grid throughout
  • 180ms ease-out transitions — fast but perceivable
  • Green accent used sparingly: only actions + status
  • Every screen = 1 primary action maximum
  • No modal interruptions — errors are always inline
  • Optimistic UI: join updates instantly, syncs silently
